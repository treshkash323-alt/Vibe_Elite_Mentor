# Elite Mentor AI (MVP v0.1)

RAG-light система для анализа веб-страниц с использованием ИИ.

## 🚀 Запуск

1. Установка зависимостей
pip install -r requirements.txt

2. Запуск сервера
uvicorn main:app --reload

3. Открыть в браузере
http://127.0.0.1:8000

## 📌 Ограничения MVP
- нет векторной базы данных
- работа только с одним контекстом
- упрощённый retrieval
- нет долговременной памяти
