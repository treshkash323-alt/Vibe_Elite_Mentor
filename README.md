# Elite Mentor AI (MVP v0.5)

Практический инструмент для анализа веб-страниц с использованием LLM.

Система загружает страницу, извлекает текст, разбивает его на чанки и отвечает на вопросы пользователя строго в рамках этого текста.

---

## 🎯 Назначение

Проект реализует MVP системы типа **RAG-light** для:

* анализа веб-страниц
* извлечения смыслов
* обучения принципам Retrieval-Augmented Generation

---

## ⚙️ Архитектура

```
User → FastAPI → HTML fetch → Text extraction → Chunking → Retrieval → LLM → Answer
```

---

## 🧠 Как это работает

1. Пользователь загружает URL страницы
2. HTML скачивается через httpx
3. Основной текст извлекается через readability + BeautifulSoup
4. Текст очищается и разбивается на чанки
5. При вопросе:

   * выбираются релевантные чанки (keyword scoring)
   * формируется контекст
   * LLM отвечает строго по нему

---

## 🚀 Функциональность MVP

* загрузка веб-страницы (ingest)
* очистка и извлечение текста
* чанкинг с overlap
* эвристический retrieval (keyword scoring)
* генерация ответа строго по контексту
* debug-режим (показ выбранных чанков)
* простой UI
* голосовой ввод (Web Speech API)

---

## ❗ Тип системы

Это **не полноценный RAG**, а:

👉 RAG-light

Причины:

* нет embeddings
* нет vector database
* нет semantic search
* retrieval на ключевых словах

---

## 📉 Ограничения

* работа только с текстом страницы
* эвристический поиск (без семантики)
* нет долговременной памяти
* данные хранятся только в RAM
* возможны неточности при сложных вопросах
* нет multi-document анализа
